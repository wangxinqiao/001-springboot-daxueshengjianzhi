# 001Springboot大学生兼职平台-后台管理

## 环境要求：
JDK8
mysql 5.7
maven
eclipse/idea

关于数据库导入
在mysql中创建数据库test,然后在数据库下执行sql文件，即parttimeplatform.sql

数据库相关配置：
打开application.yml文件，修改所有配置中数据库的ip地址、端口号、username、password等；

源码获取：[ **点此获取** ](http://www.shuyue.fun/index.php?type=productinfo&id=60)
